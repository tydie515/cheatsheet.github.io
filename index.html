Linking to a relative page.

web developers often store HTML files in the root directory or main
folder i.e. 'index.html'

  • about.html
  • contact.html
  • index.html

we can link web pages together using a 'relative path'

example code

  • <a href = "./contact.html">contact</a>

a relative path is a filename that shows the path to a local file
(a file on the same website such as 'index.html') versus an absolute path
(a full URL link HTTPS://etc... which is stored in a different folder).
